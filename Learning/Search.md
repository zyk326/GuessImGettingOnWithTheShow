#### 搜索技巧(可以组合使用)  
* 限定搜索:加英文双引号,例如限定搜索林俊觉  
```
"林俊觉"
```

* 标题关键字搜索:加**intitle:**,例如搜索芭乐情歌过时了  
```
intitle:芭乐情歌过时了
``` 

* 标题多关键字搜索:加**allintitle:**,例如搜索标题里有林俊杰和情歌  
```
allintitle:林俊杰 情歌
```
