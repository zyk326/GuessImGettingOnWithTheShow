### 2.3 数据清洗与预处理
- 清洗过的数据集简单，体现不出算法；原始数据集要花时间预处理。
- **数据增强**：人工合成
- 爬网页数据要注意**版权问题**
- 有数据后，考虑提升**数据质量**、**模型质量**、**标注质量**

### 2.4 数据标注
- **半监督**：标一些 → 训练模型 → 预测未标注 → 加入已标注 → 滚雪球（自训练）
- **找人或众包标注** / **弱监督学习**
- 把不置信的样本先拿出来训练，再放回去或许有性能提升
- 众包公司成本较低：画线 1.2 美分，画圈 2.4 美分，精度到像素画框更贵
- **主动学习和自训练**：标注一些样本 → 训练模型 → 预测样本 → 把最置信的样本加入标注 → 重复

- 推荐工具：**OpenRefine** 或 **Trifacta Wrangler**
- 数据变形 = 数据清理 + 标注 + 特征工程

### 2.6 数据变换

- **图片数据**：裁剪、下采样（需权衡数据质量和大小，不能太狠）
- **视频**：截取片段给机器

## 四、学习范式

### 4.1 监督学习
#### 组成部分
- **目标**：输入与输出是什么
- **损失（Loss）**：模型预测值与真实值的差别衡量，预测值要尽量向真实值靠近
- **模型**：优化靠近目标的过程，需要目标函数（树、线性、核方法、神经网络）
- **优化**：最小化损失

#### 决策树
- 可解释模型，能看到决策过程
- 回归 → 连续问题；分类 → 离散问题

#### 随机森林
- 从数据中随机取样本训练一棵树，重复 n 次
- 从特征中随机采样特征列
- **残差**：预测值和真实值的差（没做好的部分）
- 树模型不太需要调参

#### 线性模型
- **回归**：预测值 = 所有输入特征的加权和 + 偏移
- **分类**：数据和参数（独热）做内积 + 偏移得到置信度
- `exp(x)`：把 x 从 (-∞, +∞) 映射到正数

#### 小批量随机梯度下降（Mini-SGD）
- 损失不再变化 → 模型收敛
- ηₜ：t 时刻的学习率
- **超参数**：批量大小 b 和学习率（选取有些玄学）
- 通过优化算法最小化损失函数
- "随机"指随机采样

#### Softmax
- 先 η，再归一化（得到概率输出）

#### MLP（多层感知机）
- 包含**全连接层**
- 可学习参数：**W**（m×n，m=输出向量长度，n=输入特征维度）、**b**（偏移，长度 n）
- 隐藏层 + 激活函数 → 有一层隐藏层的 MLP
- 隐藏层参数可调 → 即"调参"
  ```
  h = relu(x @ W₁ + b₁)   # 激活函数 ReLU
  y = h @ W₂ + b₂          # 输出
  ```
- 失去激活函数 → 失去隐藏层功能，退化为线性模型
- ReLU 让模型变成**非线性**

### 4.2 半监督学习
- 标一些 → 训练模型 → 预测未标 → 加入已标注 → 滚雪球
- **有钱 → 众包 → 让人标**
- 用规则限制
- 数据质量与数量需权衡

### 4.3 无监督学习
- 数据没有标号，任务不是预测标号，主要搞**分类**

### 4.4 强化学习
- 模型与环境交互，从环境中获得观察点进行学习
- 做一些行为 → 环境给反馈 → 再进行学习
- 例：机器人走一步摔倒了 → 下一步不要走

## 五、卷积神经网络（CNN）

### 5.1 为什么需要 CNN
- 300×300 图，1000 个类，单隐藏层 MLP（隐藏层 10000）
  - 参数：90000 × 10000（约 9 亿）
- **全连接**：对所有输入元素做加权和
- **卷积**：k×k 只产生一个神经元（局部连接）
- 共享权重大幅减少参数

### 5.2 卷积层
- 输入维度：C₁×H×W（C₁=输入通道数）
- 使用 C₂ 个 k×k 卷积核
- 输出维度：C₂
- 图像是三维输入（RGB），所以用**四维表示**

### 5.3 池化层
- **最大池化（Max Pooling）**：取窗口中最大值
- **平均池化（Avg Pooling）**：取窗口中平均值
- 作用：下采样、平移不变性

### 5.4 经典网络结构
- **VGG**：内核在缩小，通道数在增加
- **NiN（Network in Network）**：
  - 用 1×1 卷积 + 全连接
  - 使用全局平均池化替代全连接（不容易过拟合）
  - 等价于把最后一层卷积变成一个 1×1 卷积输出类别数

## 六、过拟合与欠拟合

### 6.1 概念
- **过拟合**：过于关注数据细节（噪音），模型泛化能力降低
- **欠拟合**：模型复杂度不够

### 6.2 解决方案

| 问题 | 方法 |
|------|------|
| 过拟合 | 增加数据量（更多数据或数据增强）、降低模型复杂度、正则化、丢弃法（Dropout） |
| 欠拟合 | 增加模型复杂度 |

### 6.3 正则化
- 在 Loss 中加入**惩罚项**
- **L1 正则化** → 加到权重的绝对值
- **L2 正则化** → 加到权重的平方（权重衰减/Weight Decay）
- 让权重更加均匀，防止部分权重过大

### 6.4 丢弃法（Dropout）
- 在训练中随机丢弃一些神经元
- 使模型不过度依赖某些特征
- 等价于训练多个子网络

## 八、偏差与方差

### 8.1 定义
- **偏差（Bias）**：训练出来的点和真实值之间的位移
- **方差（Variance）**：每次学习的结果之间差别多大
- **期望**：希望在不同采样样本间得到的值不会很大
  - E[(y - f̂)²]
  - 样本 D = {(x₁,y₁)...(xₙ,yₙ)}，y_true = f(x) + ε（噪音）
  - x = 特征，y = 标号

### 8.2 判断
- **偏差过大** → 模型复杂度不够（欠拟合）
- **方差过大** → 模型复杂度过高（过拟合），或数据质量差

### 8.3 泛化误差分解
> 泛化误差 = 偏差² + 方差 + 数据与采集模型误差

## 九、集成学习方法

### 9.1 Bagging
- 独立（并行）训练 n 个模型
- 结果取平均或类别最多选择
- 主要为了**降低方差**（尤其模型不稳定时效果最佳）

### 9.2 Boosting
- 把多个较弱模型组合成较强模型
- 主要为了**降低偏差**
- 按顺序学习 n 个模型：
  1. 先训练一个模型
  2. 把训练好的数据拿出来，专注不好的数据
  3. 再训练
- 用较弱模型拟合**残差**

### 9.3 Stacking
- 与 Bagging 类似，把多个 base learner 放在一起
- 用于**降低方差**
- 不同模型在同一数据上训练：
  - 回归问题：每个模型输出标量 → n 维向量 → 全连接层 → 最终输出
- **炼丹方法**：第一层用 a 训练，把第一层对 b 预测结果加上 b 本身，作为第二层模型训练
- **k 折交叉验证**：每次在 k-1 上训练，验证 k；第 i 块数据在第 i 个模型上验证，其他块训练
- **多层 Stacking**：每一层在下一层基础上更好地拟合数据，降低方差，但容易过拟合
- **k 折多层 Stacking**：竞赛中常用

## 十、调参

### 10.1 基本原则
- 调参很花时间
- 从一个好的**基线**开始
- 每次只改**一个值**
- 有的超参数稳定，有的敏感
- **调参要做笔记**
- **代码要放在一个地方**
- 注意换随机种子结果浮动较大的问题

### 10.2 调参方法对比

| 方法 | 说明 |
|------|------|
| **Grid Search** | 所有超参数的排列组合试一遍，挑最好的 |
| **Random Search** ⭐最常用 | 最多试 n 次，每次随机拿一个 config，取最好 |
| **贝叶斯优化** | 学习从超参数到评估指标之间的函数 |
| **Successive Halving** | 每次选最好的一半继续训练一倍长的 epoch |
| **强者恒强** | 直接拿别人的超参数来用 |

### 10.3 模型选择
- 看模型的效果 + 跑模型的开销

## 十一、深度神经网络（DNN）

### 11.1 核心理念
- 深度神经网络是一个**编程语言**，表达对数据（结构上、理念上）的理解
- 网络里的值一开始可以不定义（可学习的权重），根据真实数据得到
- **先定义模板**，模板内容之后再学习得到
- 学习通过**可导**实现
- 定义损失函数 → 给数据 → 通过**误差反向传播**更新权重
- 深度神经网络有**多种设计模式**

### 11.2 批量归一化（Batch Normalization）
- 使用后可以使用**更大的学习率**
- 更加平滑：从 x 到 y，损失函数导数变化的平方被上限绑定（β × 距离差平方）
- 梯度改变方向被上限绑定，β 小 → 步长大 → 学习率大
- CNN 中常用（能去掉最好）
- 二维 → MLP 输入；四维 → 卷积层输入

### 11.3 层归一化（Layer Normalization）
- 用在 RNN 和 Transformer 中
- 若均值方差抖动大 → 失去标准化意义
- 区别：按哪个维度算均值和方差
- 层归一化把参数四维变二维（reshape input），如 n×c×w×h → cwh×n

## 十二、迁移学习

### 12.1 出发点
1. 一个任务上学习的模型可用来解决其他相关任务
2. 在深度学习领域特别突出（因为从头训练深度学习很贵）

### 12.2 途径
| 方式 | 说明 |
|------|------|
| 训练好模型 → 做成特征抽取模块 | 把模型当作特征提取器 |
| 训练好模型 → 直接用于另一任务 | 直接应用 |
| 训练好模型 → 微调（Fine-tuning） | 在目标任务上微调 |

### 12.3 微调（Fine-tuning）
- 固定一些层的做法很好
- **加速收敛**
- 初始点不再是随机点，更接近目标
- 越靠近最终点，损失越平滑，训练越简单
- 常用于 NLP（自然语言处理）
- **通常只需要改最后一层**

## 附：常用符号与公式
- ηₜ：t 时刻的学习率
- W：权重矩阵
- b：偏移（bias）
- ReLU：线性整流单元，`relu(x) = max(0, x)`
- softmax：先 η 再归一，得到概率输出
- E[(y - f̂)²]：期望误差
- y_true = f(x) + ε：真实值 = 函数值 + 噪音
